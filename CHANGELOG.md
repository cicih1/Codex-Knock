# Changelog

## v0.1.1

- Rename the public project name from Codex Nock to Codex Knock.
- Add the `codex_knock` module and `codex-knock` command while keeping the old `codex_nock` path compatible.
- Add English and Chinese README language switching.

- Initial release.
- Add privacy-minimal Codex event notifications.
- Add full-screen desktop popup provider.
- Add optional providers for ntfy, Bark, PushDeer, WxPusher, and Enterprise WeChat.
- Add one-command desktop setup with config backups.
